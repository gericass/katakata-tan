# katakata-tan

katakata-tan is the program that code can be input like an expert even in a programming inexperienced person.

## Requirements

- JDK 1.8
- Python 3.6.2~
- 1.1.51~
- py4j
- Processing

## Usage

1. Compile and Run `ProcessingKt.kt`
2. Run `katakata.py`
3. Type words in command line without `q`
4. Type `q`
